Deux environnements, un seul moteur

Octans s'installe sur ta machine. Tu choisis comment tu l'utilises.

Le moteur est le même dans les deux cas. Tu gardes la main sur le confort que tu veux y mettre. Aucun choix n'est irréversible.

Mode efficace

VSCode + Claude Code

L'installation minimale. Tout marche, rien de superflu.

  • Le moteur Octans tourne intégralement
  • Édition Markdown directe dans VSCode
  • Terminal intégré, Git, debug, extensions familières
  • Idéal si tu codes déjà ou si tu veux la stack la plus légère
  • Aucun setup Obsidian à apprendre

Pour qui : développeurs, profils technique, utilisateurs qui veulent une stack minimale et un workflow code-centric.

Recommandé · mode confort

VSCode + Claude Code + Obsidian

La même installation, avec un explorateur de connaissances par-dessus.

  • Tout ce que fait VSCode + Claude Code
  • Vue du graphe — tu vois tes notes se relier
  • Liens cliquables entre les notes (wikilinks)
  • Requêtes sur tes notes (Dataview) et plugins riches
  • Édition Markdown plus douce, palette de commandes, recherche avancée

Pour qui : porteurs de projet, indépendants, étudiants, entrepreneurs — tous ceux qui pensent en notes reliées plus qu'en lignes de code.

Ce que tu reçois à l'achat — quelle que soit ta config

  • Le moteur complet dans un dossier .claude/ à poser à la racine de ton projet.
  • Un guide d'installation pas-à-pas pour les deux modes (VSCode pur ou VSCode + Obsidian).
  • Une structure de dossier prête à remplir, adaptée à ton univers métier.
  • Tu peux migrer d'un mode à l'autre quand tu veux — le contenu est le même.

  • Aucun appel cloud caché. Le moteur tourne en local sur votre machine. Pas de serveur intermédiaire entre votre projet et vous.
  • Gouverné par hooks exécutables. Les seuils (profondeur, budget de questions, plafond de branches) sont enforcés mécaniquement, pas par bonne volonté.
  • Multi-prismes cloisonnés. Les décisions structurantes passent au filtre de 8 prismes de pensée indépendants — socratique, contrarian, red team, pré-mortem, faisabilité, éthicien, premiers principes, président de synthèse.
  • Vérification de cohérence continue. Le moteur boucle automatiquement entre les notes voisines de votre dossier pour détecter les contradictions de chiffres, de noms, de dates et d'engagements.
  • Garde-fou anti-hallucination intégré. Les affirmations datées sont systématiquement converties en questions pointant vers des sources officielles vérifiables.
  • Reproductible, paramétrable, versionné. Chaque composant a une version, un historique, une date de revue programmée. Aucun composant n'est figé.

État au 17/06/2026 : 40 composants production-ready · architecture v1.2 · gouvernance par 3 hooks (budget, lint, pre-commit).
