Installer votre studio IA

👋 Aucune compétence technique requise.

On installe ensemble les 5 outils gratuits dont vous avez besoin. Suivez chaque étape dans l'ordre et cochez « C'est fait » au fur et à mesure — votre progression est sauvegardée. Si vous bloquez sur une étape, l'aide est en bas de page.

⏱ ~45 min5 outils gratuitsPas à pas, à votre rythme
0 / 5 installé

1 · VSCode — votre atelier

L'éditeur gratuit de Microsoft où tout se passe.

ÉTAPE 1
  1. Cliquez sur le bouton officiel ci-dessous pour ouvrir le site de VSCode.
  2. Cliquez sur le gros bouton bleu « Download for Windows » (le téléchargement démarre tout seul).
  3. Ouvrez le fichier téléchargé (VSCodeUserSetup-….exe), acceptez les conditions, cliquez Suivant jusqu'à Installer.
  4. Cochez la case « Ajouter à PATH » si elle est proposée, puis terminez.
  5. Lancez VSCode : une fenêtre sombre s'ouvre. C'est bon. ✅
Site officiel — code.visualstudio.com Notice Windows (officielle)
📺 Tutoriel vidéo — emplacement réservé (lien YouTube à ajouter).

2 · Node.js — le moteur

Nécessaire pour faire tourner Claude Code. Installation en 2 clics.

ÉTAPE 2
  1. Ouvrez le site officiel ci-dessous.
  2. Cliquez sur le bouton « LTS » (version recommandée, la plus stable).
  3. Ouvrez le fichier node-….msi téléchargé, puis Suivant → Suivant → Installer (laissez tout par défaut).
  4. Pour vérifier : dans VSCode, menu Terminal → Nouveau terminal, tapez node --version et pressez Entrée. Un numéro s'affiche = c'est bon. ✅
Site officiel (FR) — nodejs.org/fr
📺 Tutoriel vidéo — emplacement réservé (lien YouTube à ajouter).

3 · Claude Code — votre IA développeuse

L'assistant qui écrit et édite directement dans VSCode.

ÉTAPE 3
  1. Dans VSCode, ouvrez le terminal (Terminal → Nouveau terminal).
  2. Copiez-collez cette commande, puis Entrée : npm install -g @anthropic-ai/claude-code
  3. Patientez (1–2 min). Quand la ligne se libère, tapez : claude
  4. La première fois, Claude vous demande de vous connecter avec votre compte (Pro / Max / API). Suivez le lien affiché. ✅
Documentation officielle Claude Code Choisir un plan Claude (Pro / Max)
📺 Tutoriel vidéo — emplacement réservé (lien YouTube à ajouter).

4 · Git — l'historique de sécurité

Sauvegarde chaque version de votre travail. Filet de sécurité indispensable.

ÉTAPE 4
  1. Ouvrez le site officiel ci-dessous (le téléchargement Windows démarre seul).
  2. Ouvrez le fichier Git-….exe. À chaque écran, cliquez Suivant (les réglages par défaut conviennent parfaitement).
  3. Terminez l'installation.
  4. Vérification : dans le terminal VSCode, tapez git --version. Un numéro s'affiche = c'est bon. ✅
Site officiel — git-scm.com Manuel Git (en français)
📺 Tutoriel vidéo — emplacement réservé (lien YouTube à ajouter).

5 · Obsidian — votre cerveau de notes

Là où Octans organise vos idées et décisions (le « vault »).

ÉTAPE 5
  1. Ouvrez le site officiel ci-dessous et téléchargez la version Windows.
  2. Installez (double-clic sur le fichier téléchargé), puis ouvrez Obsidian.
  3. Cliquez sur « Create new vault » (créer un coffre), donnez-lui un nom, choisissez un dossier sur votre PC.
  4. Votre coffre est prêt : c'est là que vous glisserez les fichiers Octans à l'étape suivante du parcours. ✅
Site officiel — obsidian.md Notice « créer un vault »
📺 Tutoriel vidéo — emplacement réservé (lien YouTube à ajouter).
