Objectifs Flux

L'objectif d'Octans : que vos IA et outils en ligne (LLM, automatisations, MCP) soient orchestrés depuis votre machine, sous votre contrôle. Ce schéma montre qui parle à qui — cliquez un nœud pour le détail.

Le schéma interactif

Ouvrir en plein écran pour explorer chaque nœud confortablement.

Les 3 zones, en détail
En ligne · cloud

Vos LLM & outils en ligne

Les modèles et services qui vivent sur Internet : Claude, GPT, vos automatisations cloud (n8n, Zapier…).

  • Vous gardez vos abonnements existants
  • Rien n'est verrouillé chez nous
  • Remplaçables : l'IA dessous change, pas votre méthode
MCP · le pont

Les connecteurs MCP

Le protocole qui relie vos outils en ligne à votre machine — le « pont » par lequel Octans peut actionner vos services.

  • Standard ouvert (Model Context Protocol)
  • Vous branchez ce que vous voulez
  • Chaque connecteur reste sous votre contrôle
Local · votre machine

Octans + Claude Code

Le cœur : Octans (skills, hooks, vos notes) et l'atelier IDE tournent sur votre machine. Octans instruit Claude Code, qui exécute.

  • Vos fichiers restent sur votre disque
  • Octans orchestre, Claude Code exécute
  • Vos échanges avec Claude Code vont vers Anthropic — pas par nous
Ce que le flux vous garantit

Le pilotage reste chez vous

L'orchestration vit sur votre machine. Vous décidez ce qui est branché, ce qui s'exécute, ce qui est gardé.

Vos outils en ligne, mais cadrés

Vous profitez de la puissance des LLM et automatisations cloud — sans perdre la main, parce que c'est Octans qui mène la danse.

Octans dirige, Claude exécute

Vous écrivez l'intention. Octans cadre, séquence et vérifie. Claude Code produit. Vous validez.

Des garde-fous mécaniques

Les hooks veillent en continu — cohérence, sécurité, anti-hallucination — pendant que tout ce flux tourne.

Pour installer concrètement ce flux sur votre machine, suivez la page Installer votre studio.
